CN211 Official Sites (Varied US-gov-ish Layouts)
------------------------------------------------
Each site keeps the same information structure:
- index.html (Home)
- about.html
- departments.html
- services.html
- contact.html
Plus:
- reset.css, styles.css
- logo.png (your provided logo)
- hero.jpg (your provided campus/government photo)

IMPORTANT for Packet Tracer:
- Files are FLAT (no assets folder) for maximum compatibility.
- Upload each institution folder into its own Server HTTP directory.
- Ensure index.html is the default page.

Folders:
- UDOM_SITE
- BUNGE_SITE
- MIPANGO_SITE
- CBE_SITE
- PORTAL (optional local viewing)
