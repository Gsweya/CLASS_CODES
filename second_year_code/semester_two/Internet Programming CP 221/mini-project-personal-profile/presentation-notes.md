# CP 221 Mini Project Presentation Notes (5-10 Minutes)

## 1) Page Structure
- `<!DOCTYPE html>` tells the browser the page uses HTML5.
- `<html>` is the root element containing the whole page.
- `<head>` stores metadata like page title and character encoding.
- `<body>` contains all visible content users interact with.

## 2) Content Tags
- I used `<h1>` for the main title and `<h2>/<h3>` for section/subsection hierarchy.
- `<p>` is used for normal paragraphs.
- `<strong>` emphasizes important words with strong meaning.
- `<em>` is used for stress emphasis.
- `<hr>` creates thematic breaks between major areas.
- `<br>` adds a line break inside text.

## 3) Navigation
- `<a>` creates hyperlinks.
- Some links move inside the same page using IDs (like `#contact`).
- External links open tutorials and references.
- One link uses `target="_blank"` to open in a new tab.

## 4) Media
- `<img>` embeds images with source URL in `src`.
- `alt` gives text alternatives for accessibility and image loading failures.

## 5) Lists and Table
- `<ul>` displays unordered (bulleted) items.
- `<ol>` displays ordered (numbered) steps.
- Weekly schedule table uses:
  - `<table>` container
  - `<tr>` table row
  - `<th>` header cell
  - `<td>` data cell

## 6) Forms (Focus Area)
- `<form>` groups all input controls for sending user data.
- `input type="text"` is for general text.
- `input type="email"` validates email format.
- Radio buttons allow one choice from a group.
- Checkboxes allow multiple selections.
- `<label>` connects readable text to an input for better usability/accessibility.
- `<textarea>` captures long messages.
- `<button type="submit">` sends the form data to the form `action` URL.

## Quick Demo Script
1. Open `index.html` in browser.
2. Show semantic sections: header, nav, section, footer.
3. Click navigation links.
4. Show images and explain alt text.
5. Explain schedule table rows/columns.
6. Fill form fields and click submit.
